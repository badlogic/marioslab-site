Mario's BMFG

Load a fixed-width TTF file, e.g. Tamzen 8x16r or IMB VGA 8x16. Select a font height. Right click the glyph image, "Save Image as ...", and enjoy your bitmap font.

The image consists of 16x14 glyphs, from Unicode code point 32 to code point 255. Each glyph is pixels in size.
